# Hotel Booking Application

## Overview

This project is a simple hotel booking system where users can search hotels, view rooms, and make bookings.
An admin manages all hotels and rooms.

## Features

* User login and signup
* Search hotels by location
* View hotel and room details
* Book and cancel rooms
* View booking history
* Admin can add hotels and rooms

## Tech Stack

* Frontend: React
* Backend: Node.js / Spring Boot
* Database: MySQL

## Setup Instructions

1. Clone the repository
2. Install dependencies using npm install
3. Start backend server
4. Start frontend

## Notes

* Email notifications are simulated
* Focus is on core booking functionality
